#图书馆收集整理
>目的：收集一些方便使用的公共图书馆，方便自己查询资料的使用。如果你要好的资源，也可以提交给我，不过我不会收录盗用别人资源的链接，谢谢大家了。

>原则：支持芝麻信用或者是自由注册的图书馆，多数使用的是电子资源。在办卡指南处加入芝麻信用的图书馆，方便大家注册使用。

>在上学的时候，没有用好学校的图书馆资源，等到踏入社会，需要查询一些资料的时候发现无处可寻，淘宝上的很多卖账号的，是一种解决方式，不过多数是盗用大学图书馆的，这类仅限于学校使用的，不推荐。
在这里收录了一些可以合理使用的公共图书馆，这些图书馆开放给我们方便的权限，希望大家能够珍惜，勿滥用，不要等失去之后才觉得惋惜。
